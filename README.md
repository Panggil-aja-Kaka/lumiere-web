## Overview

Lumière is a fictional minimalist skincare brand focused on gentle, effective, and honest formulas. This project was built as a freelance simulation — from client brief to final delivery — to practice real-world frontend workflows.

---

## Features

- Responsive design (Mobile, Desktop. This isn't optimized for Tablet)
- Scroll-triggered navbar with backdrop blur effect
- CSS water drop decoration
- Animated leaf decoration
- Ingredients section layout
- Swiper.js slider integration
- ScrollReveal animations
- Mobile hamburger menu with sidebar

---

## Tech Stack

- HTML5 — semantic structure
- CSS3 — BEM-inspired naming, CSS variables, mobile-first
- Vanilla JavaScript — scroll events, toggle interactions
- Swiper.js — slider/carousel
- ScrollReveal.js — scroll animations
- Anime.js — micro animations
- Remixicon — icon library
